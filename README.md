today-a-small-planet.vercel.app

このアプリは、あなたの個人情報を一切収集・保存しません。
・入力した名前や言葉は、画面の中だけに表示されます
・送信しても、サーバーにもブラウザにも記録されません
・Cookieも、ブラウザ保存（localStorage）も使っていません
・入力した内容が外部に送られることはありません
・ページを閉じれば、すべて消えます

開発者もいっさい内容を見ることはできません。
評価も、採点も、保存もしない。
ただ、あなたの想いに寄り添うだけの場所です
